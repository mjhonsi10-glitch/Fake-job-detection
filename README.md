# Fake Job Detection

AI-based detection of fake internship and job offers 
using NLP and Machine Learning.

## 📌 About
Detects fraudulent job postings using text classification
techniques with TF-IDF and ML algorithms.

## 🛠️ Technologies Used
- Python
- NLP (Natural Language Processing)
- Scikit-learn
- Pandas, NumPy
- TF-IDF Vectorization

## 📄 Research Paper
Published in **IJISRT Vol.11, Issue 5, May 2026**
